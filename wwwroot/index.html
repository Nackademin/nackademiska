Auktion

Hämta alla auktioner



get

Om en auktion är köpt med "köp nu", så sätts Sold till true.
Obs tänk på att en auktion kan vara avslutad utan något bud, då skall den ej visas i säljstatistiken.

JSON array för samtliga auktioner som innehåller:
id - Auktionens id
name - Auktionens namn
description - Auktionens beskrivning
startTime - Auktionens starttid
endTime - Auktionens sluttid
imageUrl - Auktionens bild
categoryId - Auktionens kategori id
supplierId - Auktionens leverantörs id
BuyNowPrice - Auktionens acceptpris
Sold - Om auktionen är såld eller inte

Hämta detaljerad information om en auktion



get

{id} - Auktionens id.

JSON objekt för den specifika auktionen som innehåller:
id - Auktionens id
name - Auktionens namn
description - Auktionens beskrivning
startTime - Auktionens starttid
endTime - Auktionens sluttid
imageUrl - Auktionens bild
categoryId - Auktionens kategori id
supplierId - Auktionens leverantörs id
BuyNowPrice - Auktionens acceptpris
Sold - Om auktionen är såld eller inte

Köp auktionen nu



post

JSON objekt med login information som innehåller:
auctionId - auktionens id
customerId - Kundens id

Om något gåt fel så returneras http error:
400 - Bad request

Bud

Lägga bud



post

JSON objekt med login information som innehåller:
auctionId - auktionens id
customerId - Kundens id
bidPrice - budet

Om något gåt fel så returneras http error:
400 - Bad request
OBS om du skickar in ett för lågt bud så returneras också fel.

Hämta bud för en auktion



get

{id} - Auktionens id


JSON objekt för alla bud på den specifika auktionen som innehåller:
id - Bud id
auctionId - Auktionens id
customerId - Kundens id
DateTime - DateTime för budet
BidPrice - Budet
